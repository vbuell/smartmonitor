    When internal error happens in the SmartWrapper's program flow, it collect the problem and try to investigate the source of the problem. This error code stored in the datastore and could be visualized by mtop utility.

    There are three errors codes that should tell you the type of internal problems.

    CodeDescriptionSymbol used in mtop
    CONNThe problem with connectivity (SSH).'!'
    IOProblems with file read/write operations. Output/Input file.'!'
